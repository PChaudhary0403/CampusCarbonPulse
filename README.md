## 🚀 Project Overview
This system provides an engineering-grade analytical dashboard to quantify, categorize, and interpret the greenhouse gas (GHG) emissions of the campus. Following the **GHG Protocol**, we categorize emissions into three Scopes:

- **Scope 1 (Direct)**: Diesel Generators, Canteen Fuel, and Campus Vehicles.
- **Scope 2 (Indirect)**: Grid Electricity consumption across departments and hostels.
- **Scope 3 (Value Chain)**: Student Commuting, Faculty Travel, and Supply Chain.

## 🏗️ Tech Stack
- **Backend**: Python (FastAPI / Flask) with SQLite for professional data persistence.
- **Frontend**: React (via Vite) with Tailwind CSS and Chart.js for high-fidelity visualization.
- **Design**: Premium "Dark-Green Sustainability" aesthetic with Glassmorphism.

## ⚙️ How to Run

## 📊 Key Evaluation Metrics
- **Total Footprint**: Cumulative tCO2e across all campus operations.
- **Carbon Intensity**: Emissions normalized per student.
- **Renewable Index**: Percentage contribution of rooftop solar.
- **Neutrality Pathway**: Glide path toward the 2030 Net-Zero target.
